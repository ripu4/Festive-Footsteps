# Festive Footsteps

Festive Footsteps is a platform designed to help Indian diaspora communities stay connected with Indian festivals and cultural traditions.

## Project Goal

The project aims to understand and support cultural preservation among Indians living abroad through:

- Festival guidance
- Community insights
- Survey-based data collection
- Data visualizations
- Cultural trend analysis

## Technologies Used

- HTML
- CSS

## Future Development

- Cultural Preservation Index
- Data dashboard
- Festival recommendation engine
- Survey analytics
